
# Encryption - Decryption API
It encrypts or decrypts the desired text with the specified secret text using the aes algorithm.



## Usage
#### Request
```JSON
{
    "text" : "Value"
}

```
#### Response
```JSON
{
    "text" : "request value"
    "responseText":"Decryption or Encryption Hash Value",
    "secretKey":"optional value"
}

```
